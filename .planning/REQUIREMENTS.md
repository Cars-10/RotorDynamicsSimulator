# Requirements

## V1 Requirements

### Foundation & Architecture
- **ARCH-01**: Decouple `App.tsx` state management to improve maintainability and scalability.
- **DATA-01**: Implement local export/import functionality for simulation configurations (JSON/CSV).

### UI & Visualization
- **UI-01**: Overhaul rotor and mode shape visualizations for professional, high-fidelity feel.

### Modeling Capabilities
- **MOD-01**: Add support for diverse bearing types.
- **MOD-02**: Add support for seals.
- **MOD-03**: Add support for varying material properties for shaft segments.

### Analysis Tools
- **ANA-01**: Implement Campbell diagrams.
- **ANA-02**: Implement stability maps.
- **ANA-03**: Implement transient response visualizations.
